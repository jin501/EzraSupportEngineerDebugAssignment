# Challenge documents

This folder contains the Support Engineer Debug Challenge in multiple formats.

| File | Use |
|------|-----|
| **Markdown** (see repo root) | `SUPPORT_ENGINEER_DEBUG_CHALLENGE.md` — source for editing and version control |
| **HTML** | `Support-Engineer-Debug-Challenge.html` — open in a browser to **print or save as PDF** |

## Getting the PDF

1. **From the HTML (easiest):** Open `Support-Engineer-Debug-Challenge.html` in a browser (Chrome, Safari, or Edge). Use **File → Print** and choose **Save as PDF** (or similar) as the destination.
2. **From the Markdown (if you have Pandoc):** From the repo root run:  
   `pandoc SUPPORT_ENGINEER_DEBUG_CHALLENGE.md -o docs/Support-Engineer-Debug-Challenge.pdf`

The Markdown is the single source of truth; the HTML is generated from it for easy PDF creation.
